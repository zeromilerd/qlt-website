Quantum Ledger Technology


A novel client-server cybersecurity framework deployed using Solid State System on a Chip (3SoC) powered by Zero Vulnerability Computing (ZVC) technology to deliver a Quantum resistant security to Blockchain/DLT infrastructure

SoC tech can be deployed as a client-server network (Intranet) with zero attack surface.

PROJECT MILESTONES


QLT is the result of over two decades of our work in the Cybersecurity field, which began with a 2007 patent that demonstrated a Real Online Card Key (ROCK) device that bypassed the computer OS by running a thin OS on a CD Rom to conduct an online transaction. (US Patent No. 228424B2, filed on August 12, 2002, and issued on June 5, 2007). The technological milestone got its first validation from multiple EU Consortiums that include 6 Cybersecurity Centres of Excellence. The Zero Vulnerability Computing technology that eventually took the shape of QLT, got its first real-world validation by winning an H2020 NGI grant. FAQs

Zero Vulnerability Computing (ZVC) is an encryption-agnostic cybersecurity framework that aims to provide a zero vulnerability and zero attack surface environment for the exchange of information between computers. It is designed to completely obliterate the attack surface of the computing devices, making them robust and energy-efficient.

The 3SoC refers to the Solid State Software on a Chip, which is a key component of the Zero Vulnerability Computing (ZVC) framework. It is a compact system that integrates all the conventional layers of firmware drivers, operating system, and application layer into a secure and energy-efficient chip. The 3SoC is designed to have zero attack surface, making it resilient to cyber threats. In the context of the Quantum-safe Ledger Technology (QLT) framework, the 3SoC servers are used to host blockchain nodes, creating a secure 3SoC tunnel for accessing blockchain infrastructure. Additionally, the 3SoC client device is used as a hardware wallet to securely connect to the blockchain nodes, ensuring a high level of security for blockchain and cryptocurrency transactions.

The 3SoC achieves zero vulnerability by integrating all the conventional layers of firmware drivers, operating system, and application layer into a secure and energy-efficient chip. This integration results in a compact Solid State Software on a Chip that has zero attack surface, making it resilient to cyber threats. By consolidating these layers into a single secure chip, the 3SoC minimizes the potential points of vulnerability that could be exploited by malicious actors. This approach aims to provide a robust and secure environment for computing and communication, particularly in the context of blockchain and cryptocurrency transactions.

QLT, or Quantum-safe Ledger Technology, is an innovative approach to securing blockchain and cryptocurrency transactions from potential quantum computer threats. It is based on the Zero Vulnerability Computing (ZVC) framework, which provides a secure environment for information exchange. QLT segregates blockchain infrastructure from the mainstream computing environment, creating a secure 3SoC tunnel for accessing blockchain nodes. This platform-agnostic framework is cost-effective, resource-efficient, and scalable, making it a promising solution for enhancing blockchain and cryptocurrency security.

QLT offers platform agnosticism, cost-effectiveness, resource efficiency, scalability, and regulatory compliance to the blockchain industry. It presents a comprehensive and adaptable solution for enhancing the security and resilience of blockchain and cryptocurrency infrastructures.
